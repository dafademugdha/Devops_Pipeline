# DevOps Microservices Project

## 📌 Problem Statement
This project is a simple Issue Reporting System where users can:
- Register/Login
- Report issues (like potholes, garbage, etc.)
- Admin can update issue status

## 🏗️ Architecture
- User Service (Port 3002)
- Issue Service (Port 3001)
- Admin Service (Port 3003)
- Docker + Docker Compose
- CI/CD using GitHub Actions

## ⚙️ Tech Stack
- Node.js
- Express.js
- Docker
- GitHub Actions

## 🚀 How to Run
```bash
docker-compose up --build
